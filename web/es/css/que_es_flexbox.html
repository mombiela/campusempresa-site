Introducción a Flexbox

Flexbox, o "Flexible Box Layout", es un modelo de diseño en CSS que facilita la creación de estructuras de diseño complejas y adaptables. Flexbox permite alinear y distribuir espacio entre los elementos de un contenedor de manera eficiente, incluso cuando su tamaño es desconocido o dinámico.

  • Características principales de Flexbox:
  • Proporciona un diseño unidimensional, alineando elementos en una sola dirección: fila o columna.
  • Facilita la alineación de elementos y la distribución de espacio entre ellos.
  • Ofrece mayor control sobre el tamaño y el espacio de los elementos dentro de un contenedor flexible (flex container).

Componentes de Flexbox

  • Flex Container:

  • El elemento contenedor que utiliza Flexbox. Se define con la propiedad display: flex; o display: inline-flex;.

  • Flex Items:

  • Los elementos hijos directos del contenedor flexible, que se disponen y alinean utilizando Flexbox.

Propiedades del contenedor Flexbox

  • display:

  • Define el elemento como un contenedor flexible.

  • Ejemplos: display: flex;, display: inline-flex;

  • flex-direction:

  • Define la dirección principal en la que se colocan los elementos flexibles.

  • Valores: row, row-reverse, column, column-reverse

  • flex-wrap:

  • Controla si los elementos flexibles se envuelven o no dentro del contenedor.

  • Valores: nowrap, wrap, wrap-reverse

  • justify-content:

  • Define cómo se distribuye el espacio entre los elementos a lo largo de la dirección principal.

  • Valores: flex-start, flex-end, center, space-between, space-around, space-evenly

  • align-items:

  • Define cómo se alinean los elementos a lo largo del eje transversal (perpendicular al eje principal).

  • Valores: flex-start, flex-end, center, baseline, stretch

  • align-content:

  • Define cómo se distribuye el espacio entre las líneas cuando hay múltiples líneas de elementos flexibles.

  • Valores: flex-start, flex-end, center, space-between, space-around, stretch

Ejemplo básico de Flexbox

.container {
display: flex;
flex-direction: row;
justify-content: space-between;
align-items: center;
}
.item {
background-color: lightcoral;
padding: 10px;
margin: 5px;
}

En este ejemplo, un contenedor flexible tiene elementos distribuidos en una fila con espacio entre ellos, y los elementos se alinean al centro del eje transversal.
