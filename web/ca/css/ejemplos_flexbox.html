Los ejemplos prácticos de Flexbox muestran cómo se pueden utilizar las propiedades de Flexbox para crear diseños flexibles y adaptables. A continuación, se presentan algunos ejemplos comunes:

Ejemplo 1: Centro de un elemento en un contenedor

.container {
display: flex;
justify-content: center;
align-items: center;
height: 100vh;
background-color: lightgray;
}
.item {
background-color: lightcoral;
padding: 20px;
}

En este ejemplo, el ítem se centra horizontal y verticalmente dentro del contenedor usando justify-content: center; y align-items: center;.

Ejemplo 2: Diseño de barra de navegación

.nav {
display: flex;
justify-content: space-between;
background-color: darkblue;
padding: 10px;
}
.nav a {
color: white;
text-decoration: none;
padding: 10px;
}

En este ejemplo, una barra de navegación con enlaces se distribuye uniformemente utilizando justify-content: space-between;.

Ejemplo 3: Tarjetas con altura y ancho iguales

.container {
display: flex;
flex-wrap: wrap;
justify-content: center;
}
.card {
flex: 1 1 200px;
margin: 10px;
background-color: lightblue;
padding: 20px;
}

En este ejemplo, las tarjetas dentro del contenedor se envuelven en varias líneas si es necesario y cada tarjeta tiene un ancho mínimo de 200px utilizando flex: 1 1 200px;.

Ejemplo 4: Diseño de pie de página con distribución espacial

.footer {
display: flex;
justify-content: space-around;
background-color: black;
color: white;
padding: 20px;
}
.footer div {
padding: 10px;
}

En este ejemplo, los elementos dentro del pie de página se distribuyen uniformemente utilizando justify-content: space-around;.

Ejemplo 5: Columna de igual altura en diseño de tres columnas

.container {
display: flex;
}
.column {
flex: 1;
background-color: lightgreen;
margin: 5px;
padding: 20px;
}

En este ejemplo, tres columnas se crean dentro de un contenedor flexible, todas con igual altura y ancho utilizando flex: 1;.
