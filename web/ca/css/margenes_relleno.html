  • Propiedades de márgenes

  • margin: Define el margen completo en una sola declaración. Ejemplo: margin: 10px;

  • margin-top, margin-right, margin-bottom, margin-left: Permiten definir márgenes individuales para cada lado. Ejemplo: margin-top: 20px;

  • margin:auto: Centra un elemento horizontalmente dentro de su contenedor. Ejemplo: margin: 0 auto;

  • Propiedades de relleno

  • padding: Define el relleno completo en una sola declaración. Ejemplo: padding: 15px;

  • padding-top, padding-right, padding-bottom, padding-left: Permiten definir rellenos individuales para cada lado. Ejemplo: padding-left: 10px;

Ejemplo de uso de propiedades de márgenes y relleno

div {
margin: 20px;
padding: 15px;
}
p {
margin-top: 10px;
margin-bottom: 10px;
padding-left: 5px;
}

Este ejemplo define un div con un margen de 20px alrededor y un relleno de 15px en todos los lados. Para un párrafo (p), establece un margen superior e inferior de 10px y un relleno izquierdo de 5px.
