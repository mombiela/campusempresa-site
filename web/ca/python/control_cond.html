Las estructuras condicionales en Python permiten ejecutar bloques de código específicos basados en condiciones dadas. Estas estructuras son fundamentales para la toma de decisiones en programación.

Sentencia if

La sentencia if evalúa una condición. Si la condición es verdadera (True), se ejecuta el bloque de código indentado bajo el if.

x = 10
if x > 5:
print("x es mayor que 5")

En este ejemplo, x es mayor que 5, por lo tanto, la condición es verdadera y se imprime "x es mayor que 5".

Sentencia elif

La sentencia elif (abreviatura de "else if") permite verificar múltiples condiciones. Si la condición if inicial es falsa, Python evalúa la siguiente condición elif.

x = 10
if x > 15:
print("x es mayor que 15")
elif x > 5:
print("x es mayor que 5 pero menor o igual a 15")

En este caso, la primera condición if es falsa (x no es mayor que 15), pero la segunda condición elif es verdadera (x es mayor que 5), por lo que se imprime "x es mayor que 5 pero menor o igual a 15".

Sentencia else

La sentencia else se usa para ejecutar un bloque de código si todas las condiciones anteriores (if y elif) son falsas.

x = 3
if x > 15:
print("x es mayor que 15")
elif x > 5:
print("x es mayor que 5 pero menor o igual a 15")
else:
print("x es 5 o menor")

Aquí, ambas condiciones (if y elif) son falsas, por lo que se ejecuta el bloque else y se imprime "x es 5 o menor".

Ejemplo Completo

Un ejemplo completo utilizando if, elif, y else:

edad = int(input("Introduce tu edad: "))

if edad < 18:
print("Eres menor de edad")
elif edad < 65:
print("Eres adulto")
else:
print("Eres mayor de 65 años")

En este ejemplo:

Se solicita al usuario que introduzca su edad.

Si la edad es menor que 18, se imprime "Eres menor de edad".

Si la edad es mayor o igual a 18 pero menor que 65, se imprime "Eres adulto".

Si la edad es 65 o mayor, se imprime "Eres mayor de 65 años".

Uso de operadores lógicos en condiciones

Puedes combinar múltiples condiciones en una sola sentencia if usando operadores lógicos como and, or, y not.

x = 20
if x > 10 and x < 30:
print("x está entre 10 y 30")

En este ejemplo, ambas condiciones (x > 10 y x < 30) deben ser verdaderas para que el bloque de código se ejecute.

Importancia de la indentación

Es crucial recordar que Python usa la indentación para definir bloques de código. Asegúrate de que todas las líneas de código dentro de una estructura condicional estén correctamente indentadas.

if x > 5:
print("x es mayor que 5")
if x > 10:
print("x es también mayor que 10")
print("Esta línea se ejecuta si x es mayor que 5, independientemente de x > 10")

En resumen, las estructuras condicionales (if, elif, else) en Python permiten ejecutar diferentes bloques de código basados en condiciones, facilitando la toma de decisiones en tus programas. La correcta comprensión y uso de estas estructuras es esencial para cualquier programador.
