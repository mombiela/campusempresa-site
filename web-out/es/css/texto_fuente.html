  • Propiedades de color del texto

  • color: Establece el color del texto. Ejemplo: color: red;

  • text-decoration: Define decoraciones adicionales como subrayado, línea a través, etc. Ejemplo: text-decoration: underline;

  • text-transform: Controla la capitalización del texto. Ejemplo: text-transform: uppercase;

  • Propiedades de alineación del texto

  • text-align: Alinea el texto dentro de su contenedor. Valores comunes: left, right, center, justify. Ejemplo: text-align: center;

  • text-indent: Indenta la primera línea del texto en un contenedor. Ejemplo: text-indent: 20px;

  • line-height: Ajusta la altura de las líneas de texto. Ejemplo: line-height: 1.5;

  • Propiedades de espacio y espaciado del texto

  • letter-spacing: Ajusta el espacio entre caracteres. Ejemplo: letter-spacing: 2px;

  • word-spacing: Ajusta el espacio entre palabras. Ejemplo: word-spacing: 4px;

  • Propiedades de fuentes

  • font-family: Especifica la familia de fuentes para el texto. Ejemplo: font-family: Arial, sans-serif;

  • font-size: Define el tamaño del texto. Ejemplo: font-size: 16px;

  • font-style: Establece el estilo de la fuente, como //cursiva//. Ejemplo: font-style: italic;

  • font-weight: Define el grosor del texto. Valores comunes: normal, bold, bolder, lighter. Ejemplo: font-weight: bold;

Ejemplo de uso de propiedades de texto y fuentes

p {
color: blue;
text-align: center;
text-transform: uppercase;
letter-spacing: 1px;
font-family: "Times New Roman", serif;
font-size: 18px;
font-style: italic;
font-weight: bold;
}

Este ejemplo define un párrafo (p) con texto azul, centrado, en mayúsculas, con espacio entre letras de 1px, utilizando la fuente "Times New Roman" con un tamaño de 18px, estilo //cursiva// y grosor negrita.
