  • Introducción Los selectores en CSS permiten a los desarrolladores apuntar a elementos específicos dentro de un documento HTML para aplicarles estilos. Existen varios tipos de selectores, cada uno con sus propias características y usos.

  • Selectores básicos startalert

  • Selector de tipo: Selecciona todos los elementos de un tipo específico.

p {
color: blue;
}
  • Selector de clase: Selecciona todos los elementos que tienen una clase específica.
.intro {
font-size: 14px;
color: green;
}
  • Selector de ID: Selecciona un único elemento que tiene un ID específico.
#header {
background-color: yellow;
}
  • Selector universal: Selecciona todos los elementos en la página.
* {
margin: 0;
padding: 0;
}
  • Selector de atributo: Selecciona elementos que tienen un atributo específico.
a[href] {
color: red;
}

endalert

  • Selectores de combinadores startalert
  • Selector descendiente: Selecciona elementos que son descendientes de otro elemento.
div p {
color: blue;
}
  • Selector hijo: Selecciona elementos que son hijos directos de otro elemento.
div > p {
color: green;
}
  • Selector de adyacente inmediato: Selecciona el elemento que está inmediatamente después de otro.
h1 + p {
margin-top: 0;
}
  • Selector de hermanos generales: Selecciona todos los elementos que son hermanos de un elemento especificado.
h1 ~ p {
color: red;
}

endalert

  • Selectores de pseudoclases startalert
  • :hover: Selecciona elementos cuando el usuario pasa el cursor sobre ellos.
a:hover {
color: red;
}
  • :first-child: Selecciona el primer hijo de su elemento padre.
p:first-child {
font-weight: bold;
}
  • :nth-child(n): Selecciona el enésimo hijo de su elemento padre.
li:nth-child(2) {
color: blue;
}
  • :not(selector): Selecciona todos los elementos que no coinciden con el selector dado.
:not(p) {
color: gray;
}

endalert

  • Selectores de pseudoelementos startalert
  • ::before: Inserta contenido antes del contenido de un elemento.
p::before {
content: "Nota: ";
color: red;
}
  • ::after: Inserta contenido después del contenido de un elemento.
p::after {
content: " (Leído)";
color: blue;
}
  • ::first-letter: Selecciona la primera letra de un elemento.
p::first-letter {
font-size: 2em;
color: green;
}
  • ::first-line: Selecciona la primera línea de un elemento.
p::first-line {
font-weight: bold;
}

endalert

  • Conclusión Conocer los distintos tipos de selectores en CSS es crucial para aplicar estilos de manera eficiente y precisa. Cada tipo de selector tiene su propósito y uso específicos, permitiendo una gran flexibilidad en el diseño y desarrollo de sitios web.
